Student Performance Prediction Using Machine Learning

Overview

This project predicts student exam scores using machine learning techniques. The dataset contains academic, personal, and environmental factors such as attendance, study hours, sleep hours, previous scores, family income, and parental involvement.

Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

Machine Learning Models

1. Linear Regression
2. Random Forest Regression

Results

- Linear Regression R² Score: 0.770
- Random Forest R² Score: 0.647

Key Findings

The most important factors affecting exam scores were:

- Attendance
- Hours Studied
- Previous Scores
- Tutoring Sessions
- Sleep Hours

Conclusion

Linear Regression performed better than Random Forest for this dataset and was able to explain approximately 77% of the variation in student exam scores.
